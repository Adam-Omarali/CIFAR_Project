Creating a Non-Binary AI Image Classifier

Creating an AI to Classify the CIFAR-10 Dataset

Computers are able to focus and work way better than us, now just give them some sight and they can do things way better than us. I decided to use Convolutional Neural Networks to help the computer see the images in the dataset CIFAR-10, which is comprised of 5 different types of vehicles and 5 types of animals. I trained my network to see if it could label an image correctly given a random image of a animal or vehicle in the dataset. This is the first large step in giving computers non binary information and the applications are endless. From medical imaging to species data collection, the possibilities are endless using CNNs!

How It Works

01. Importing the Dataset

First, the dataset must be imported and fixed for optimal use. This means changing to black and white in this case, since it allows Kernal's to read the images with the same quality even though there is less information.

02. Designing the Network

I used tesnserflow and keras to use layers like convolutional, dense, flatten, max pooling and dropout to allow the AI to gather information from the dataset and learn how to classify the images.

03. Training the Network

Finally, we allow the network to train using a portion of the dataset and then test it with the remaing portion of the dataset. Overall my model had an accuracy of 84%!
